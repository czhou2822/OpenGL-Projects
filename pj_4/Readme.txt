This code takes buddha-ply file and demostrate different texture and lights via OpenGL.  

Key A:
Toggle shadow
Key B:
Toggle texture
Key C:
Toggle Reflection
Key D:
Toggle Refration
Key E:
draw bounding box
Key P/p:
Increase/decrease spotlight cut off angle by a small amount
Key S/s:
Increase/decrease the shininess of the Buddha by a small amount
Key I/i:
Increase/decrease the index of refraction of the Buddha by a small amount 
Key F/f:
increase/decrease fog volunmn
